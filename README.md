# Life Insurance — Mesa Group Consulting

Standalone prototype for the **Life Insurance** landing page.

Helps families compare and apply for term and whole life insurance policies through a multi-carrier marketplace, with coverage calculator and instant quotes.

## Tech Stack
React 18 · TypeScript · Vite · Tailwind CSS · shadcn/ui
